## üéì Key Concepts Explained

### Negative Lookbehind: `(?<!Year\s)`
Prevents matching if "Year " comes before the pattern.
Prevents "Model" in "Year Model: 2025" from being captured as series.

### Priority Ordering
Year Model pattern must run BEFORE Series pattern.
This prevents year from being captured as model name.

### 4-Digit Validation: `/^\d{4}$/`
Checks if a value is exactly 4 digits (a year).
Used as fallback validation if regex captures a year.

### Text Preprocessing
Cleans OCR artifacts (pipes, colons) before pattern matching.
Improves regex reliability and field extraction.

### Compound Labels
Support for Philippine LTO labels like "Make/Brand", "Series/Model".
Each variation is explicitly included in regex patterns.
